# pyproject_DITBiblio
ceci est un projet de classe pour le cours de python, il s'agit d'implement des notions de POO pour la gestion d'une bibliotheque numerique


Structure du projet

pyproject_DITBiblio/
│
├── data/       # Dossier pour la persistance (fichiers .txt ou .json)
├── logs/       # Journal des actions
├── models.py   # Définition des classes
├── manage.py  # Logique métier (ajouter, emprunter, statistiques)
├── main.py    # Point d'entrée de l'application (Menu interactif)
└── README.md  # Documentation du projet
